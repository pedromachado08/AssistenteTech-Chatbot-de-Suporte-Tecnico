# AssistenteTech-Chatbot-de-Suporte-Tecnico
O TechSupportIA é um chatbot de suporte técnico que analisa a pergunta do usuário, normaliza o texto e compara com uma base de conhecimento para encontrar a resposta mais adequada. Caso não exista correspondência suficiente, o sistema oferece abrir um chamado e gera um protocolo automático para suporte humano.
